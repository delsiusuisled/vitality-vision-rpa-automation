# Vitality Vision RPA Automation

## 📌 Project Overview
This project is an end-to-end Robotic Process Automation (RPA) solution built using UiPath Studio.  
It automates customer data extraction, validation, BMI calculation, and personalized email distribution using structured workflow design and exception handling.

The automation improves efficiency, reduces manual processing errors, and ensures accurate rule-based validation before email communication.

---

## 🚀 Key Features

- 🔍 Customer data extraction from structured data source
- 🧮 Automated BMI calculation logic
- ✅ Rule-based customer validation
- 📧 Automated personalized email workflow
- ⚠ Structured exception handling (BusinessRuleException & Try/Catch)
- 🗂 Modular workflow architecture
- 📝 Execution logging for traceability
- 🔄 Git version control integration

---

## 🛠 Technologies Used

- UiPath Studio
- .NET Framework
- Git & GitHub (Version Control)
- Structured DataTables
- Exception Handling Mechanisms

---

## 🧠 Automation Logic Flow

1. Load configuration settings
2. Extract customer records
3. Validate data fields sequentially
4. Perform BMI computation
5. Separate valid and invalid customers
6. Send personalized promotional emails
7. Log execution results
